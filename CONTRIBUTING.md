# Contribution Guide

We :heart: pull requests!

To contribute:

- Fork the repo
- Run `composer install`
- Add a functional tests to ensure Laravel can load the package
- Add your bridging service provider
- Run your tests
- Update composer.json, add the package to the suggested packages
- Update README.md
